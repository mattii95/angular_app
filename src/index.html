Qué es HTML?

  1. Describe la ESTRUCTURA y el CONTENIDO de una página web.
  2. Consiste en etiquetas.
  3. Las etiquetas le dicen al navegador cómo mostrar el contenido.
  4. Es un lenguaje y como tal tiene su vocabulario (palabras), su gramática (reglas).
  5. En la actualidad el HTML está definido por el W3C.
